# Skill: Create Premium Page

Use this skill when creating any major user-facing page.

## Goal

Create a premium, modern, responsive, conversion-focused page.

## Steps

1. Understand the page goal.
2. Define the main CTA.
3. Create clear visual hierarchy.
4. Use mobile-first layout.
5. Add premium spacing and typography.
6. Add subtle animations.
7. Add loading, empty, and error states if data-driven.
8. Optimize images.
9. Make CTA visible and easy to use.
10. Review for performance.

## UI Requirements

- Use Tailwind CSS.
- Use shadcn/ui where suitable.
- Use responsive grids.
- Use modern cards.
- Use subtle shadows and borders.
- Use clean section spacing.
- Use consistent heading sizes.
- Avoid clutter.

## Animation Guidelines

Use animation for:

- Hero section entrance
- Product card hover
- Filter panel open/close
- Section reveal
- CTA hover
- Image transitions

Avoid:

- Slow animations
- Distracting effects
- Heavy motion on every element
- Layout-shifting animations

## Final Check

The page should feel custom-built, not like a generic template.
