# Skill: Build Product Filtering

Use this skill when building smart search and filters for product discovery.

## Goal

Allow users to quickly find tyres without endless scrolling.

## Filters

Support filters such as:

- Keyword search
- Tyre size
- Vehicle type
- Brand
- Price range if price is available
- Category
- Application
- Ply rating / star rating if useful

## UX Rules

- Filters must work well on mobile.
- Use a filter drawer on small screens.
- Show active filters clearly.
- Allow clearing filters.
- Keep search fast.
- Debounce keyword input if needed.
- Preserve filters in URL query params where useful.
- Show empty state when no products match.

## Product Listing UX

- Product cards should be easy to scan.
- Important tyre specs should be visible.
- Enquiry CTA should be clear.
- Save button should be available without login.
- Compare option should appear only if comparison feature is enabled.

## Final Check

A user should be able to find the correct tyre in seconds.
