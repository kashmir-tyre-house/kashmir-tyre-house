# Skill: Add Animations

Use this skill when improving UI motion and micro-interactions.

## Goal

Add smooth, premium animations without hurting performance.

## Good Animation Use Cases

- Page section reveal
- Product card hover
- Image hover zoom
- Filter drawer open/close
- Wishlist save feedback
- Enquiry item add/remove feedback
- Admin table row actions
- Form success state

## Rules

- Keep animations subtle.
- Prefer CSS transitions for simple effects.
- Use transform and opacity.
- Avoid animating width, height, top, left when possible.
- Keep durations short.
- Respect reduced-motion preferences.
- Do not animate everything.

## Suggested Motion Style

- Duration: 150ms to 300ms
- Easing: smooth and natural
- Hover: slight lift, soft shadow, image scale
- Page entrance: fade + small translate
- CTA: subtle scale or glow

## Final Check

Animations should make the interface feel premium, not playful or distracting.
