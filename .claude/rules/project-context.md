# Rule: Project Context

This project is Kashmir Tyre House Tyre Product Enquiry Platform.

It is a product enquiry website, not a full ecommerce store.

## Core Scope

Frontend users can:

- Browse tyre products
- Search and filter products
- Save products without login
- Select one or multiple products
- Submit enquiry form
- View company/about information
- View technical tyre advisory and site services

Admins can:

- Login securely
- Add, update, delete products
- Manage product images
- Manage About Us images
- Optionally view/manage enquiries

## Important

Frontend users must not be required to login.

The platform should be fast, simple, premium, and conversion-driven.
