# Rule: UI/UX Quality

The UI must be top-notch.

## Required UI Quality

Every user-facing page should feel:

- Premium
- Modern
- Professional
- Trustworthy
- Fast
- Clean
- Conversion-focused

## Design Principles

- Mobile-first.
- Clear visual hierarchy.
- Strong CTAs.
- Clean spacing.
- Good typography.
- Premium product cards.
- Smooth but subtle animations.
- High-quality image presentation.
- No clutter.
- No generic template feel.

## Product Discovery UX

Users should not scroll endlessly.

Include:

- Search by keyword
- Filter by tyre size
- Filter by vehicle type
- Filter by brand
- Filter by category/application where useful
- Clear active filters
- Clear all filters button
- Empty state when no result is found

## Enquiry UX

The enquiry flow should be very clear.

- Users can select single or multiple products.
- Selected products should be visible.
- Enquiry CTA should be obvious.
- Form should be short and easy.
- Confirmation state should be polished.

## Animation Rules

Use animations for polish, not decoration.

Allowed:

- Section reveal
- Product hover lift
- Image hover zoom
- Filter drawer transition
- Wishlist save feedback
- CTA hover
- Form success animation

Avoid:

- Heavy animations
- Slow transitions
- Distracting effects
- Animation that harms performance
