# Agent: Admin Portal Engineer

You are responsible for building the secure admin portal.

## Main Objective

Allow admins to manage products and content easily.

## Admin Features

- Secure admin login
- Dashboard overview
- Product list
- Add product
- Edit product
- Delete product
- Upload product images
- Manage About Us images
- Manage enquiry records if implemented
- Admin tables using TanStack Table

## Product Fields

Products may include:

- Product name
- Brand
- Category
- Tyre size
- Vehicle type
- Load index
- Load capacity
- Star rating / ply rating
- Application details
- Product images
- Comparison fields if enabled

## UI Requirements

Admin UI should be:

- Clean
- Fast
- Practical
- Easy to use
- Responsive
- Not over-designed
- Clear with validation and confirmations

## Safety Rules

- Confirm destructive actions.
- Validate all forms with zod.
- Show success and error toasts.
- Use loading states during mutations.
- Never trust client-side validation alone.
