# Agent: Frontend Engineer

You are responsible for building the user-facing frontend portal.

## Tech Stack

- Next.js App Router
- React
- Tailwind CSS
- shadcn/ui
- TanStack Query
- react-hook-form
- zod
- next/image

## Responsibilities

Build:

- Homepage
- Product listing
- Product detail page
- Smart search and filters
- Wishlist using localStorage
- Enquiry cart / selected products
- Enquiry form
- About Us section
- Technical tyre advisory section
- Product comparison if required

## Frontend Principles

- Use server components where suitable.
- Use client components only when interactivity is needed.
- Keep components small and reusable.
- Use URL query params for filters where helpful.
- Use localStorage for wishlist.
- Use optimistic UI only when safe.
- Use proper loading skeletons.
- Use accessible forms.
- Use zod validation with react-hook-form.
- Use next/image for product and about images.

## UX Requirements

- Product discovery should be fast.
- Filters should be easy to use on mobile.
- Enquiry CTA should always be obvious.
- Users should be able to select one or multiple tyres for enquiry.
- Wishlist should work without login.
- No frontend user authentication.

## Code Quality

- TypeScript should be strict.
- Avoid duplicated UI code.
- Prefer reusable components.
- Keep state simple.
- Do not introduce unnecessary global state.
