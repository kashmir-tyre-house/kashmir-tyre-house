# Skill: Optimize Performance

Use this skill when reviewing or improving performance.

## Goal

Make the app fast, smooth, and production-ready.

## Checklist

- Use Next.js Server Components by default.
- Use Client Components only for interactivity.
- Optimize images with next/image.
- Avoid large client bundles.
- Lazy-load non-critical components.
- Use proper loading skeletons.
- Avoid layout shift.
- Cache data where appropriate.
- Paginate product/admin lists if they grow.
- Avoid unnecessary re-renders.
- Keep animations GPU-friendly.
- Check mobile performance.

## Image Rules

- Use next/image.
- Provide width and height.
- Use proper sizes attribute.
- Compress images.
- Use R2 for storage.
- Avoid loading full-size images in cards.

## Final Check

The app should feel fast even on average mobile devices.
